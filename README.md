# WeatherNow Premium

**Website:** [https://your-vercel-domain.vercel.app](https://your-vercel-domain.vercel.app)  

WeatherNow Premium adalah aplikasi web **satu halaman (one-page)** untuk menampilkan informasi cuaca secara realtime, lengkap dengan **forecast 3 hari**, **chart suhu**, **unit converter**, **dark mode**, dan **history pencarian**.  

## Teknologi yang Digunakan
- **HTML & JavaScript** – Struktur dan interaktivitas halaman
- **Tailwind CSS** – Styling cepat dan responsif
- **Chart.js** – Visualisasi grafik suhu (line chart)
- **OpenWeatherMap API** – Mendapatkan data cuaca realtime dan forecast
- **Vercel** – Platform hosting dan deployment gratis

## Fitur Utama
1. **Pencarian Cuaca Realtime** – Masukkan nama kota dan dapatkan informasi cuaca saat ini
2. **Forecast 3 Hari** – Menampilkan suhu, deskripsi cuaca, dan probabilitas hujan untuk hari ini, besok, dan lusa
3. **Chart Suhu** – Grafik line chart yang menunjukkan perubahan suhu forecast
4. **Unit Converter** – Ganti antara Celsius (°C) dan Fahrenheit (°F)
5. **Dark Mode** – Mode gelap yang merubah warna card dan teks, nyaman di malam hari
6. **History Pencarian** – Menyimpan riwayat kota yang pernah dicari, bisa di-expand untuk melihat detail cuaca saat itu
7. **Responsive Design** – Tampilan rapi untuk desktop maupun mobile
