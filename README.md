# WeatherNow

WeatherNow adalah aplikasi web **satu halaman (1-page)** untuk melihat informasi cuaca secara realtime.  
Project ini menggunakan **HTML, Tailwind CSS, dan JavaScript** saja, tanpa backend, sehingga bisa langsung dijalankan di browser.

## Fitur

- Cari cuaca kota apa saja secara realtime menggunakan **OpenWeatherMap API**  
- Background berubah sesuai cuaca (cerah, awan, hujan, badai)  
- Animasi suhu naik dari 0°C ke suhu aktual  
- Icon cuaca animasi floating  
- Menampilkan **waktu lokal kota**  
- History pencarian interaktif:  
  - Default hide, baru muncul setelah pencarian pertama  
  - Klik kota di history → tampilkan detail cuaca saat itu (suhu, deskripsi, kelembapan, angin, timestamp)
